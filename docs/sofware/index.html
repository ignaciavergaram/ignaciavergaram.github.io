Introducción

En esta sección verán cómo realizar un filtro simple para Instagram, donde podrán agregarle un “efecto 3D” a un plano. En este caso, se escogió un cuadro típico del movimiento artístico del “Pop Art”, del artista Lichtenstein.

Este proyecto fue realizado para el ramo “Entornos Virtules y Experiencias Inmersivas” del segundo semestre del año 2020, para el cual realicé un PDF explicativo con un paso a paso del proceso en el programa Spark AR, además de un video tipo making of mostrando todo el proceso previo hasta llegar a la etapa de ocupar este programa para la creación del filtro.

Spark AR Studio

Spark AR Studio es un programa creado por Facebook para la realización de filtros para la plataforma de Instagram o Facebook, utilizando realidad aumentada. Este programa posee una infinidad de opciones para crear filtros de distintos tipos, ya sea para filtros que funciones sobre el rostro, sobre superficies, interactivos tipo trivia, etc.

Este software es gratuito y tiene la ventaja de que hay muchos tutoriales en YouTube e incluso cursos gratuitos en la misma página de Spark AR.

Link de descarga:

https://sparkar.facebook.com/ar-studio/download/

Spark AR Player

Spark AR Player es una aplicación para el celular, que se utiliza para probar los filtros creados previamente en Spark AR Studio.

Paso 1: Descargar Spark AR Player desde la tienda de aplicaciones de tu celular y conectarlo mediante cable USB al computador con el programa Spark AR Studio abierto.

Paso 2: Una vez descargado, abrir Spark AR Player.

Paso 3: Habilitar el uso de la cámara y micrófono mientras se ocupa la aplicación.

Paso 4: Desde el computador, fijarse en el panel izquierdo de Spark AR Stupio y apretar la opción “Test on Device”, luego apretar “Send” abajo donde aparece el nombre de tu celular y después sale “Preview on Spark AR Player”.
